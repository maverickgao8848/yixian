# 产品需求文档 (PRD) - 西递秘档：明经遗梦 (MVP 视觉验证版)

## 1. 背景与目标 (Background & Goals)
- **是什么？** 解决目前西递景区传统导览方式枯燥、导游资源不足的问题，开发一款“线上解谜+线下打卡”的微信小程序，起到**数字化辅助导游**的作用。
- **业务目标**：通过游戏化手段提升游客参与度、游览趣味性和停留时间。
- **本次迭代目标**：快速开发出一个敏捷验证版本，重点跑通**前端基础视觉、核心交互逻辑（序章+第一幕）**，以便快速评估美术资产表现和用户操作流。

## 2. 目标用户 (Target Audience)
- **用户画像**：普通游客、亲子家庭、初次到访景区的年轻人。无专业建筑/历史背景。
- **核心诉求**：无需复杂动脑、低门槛、高容错、视觉好看且有成就感。

## 3. 需求范围 (Scope)
### **In Scope (本次覆盖必做)**
1. **入场页 (Entry Page)**：展示美术背景素材，提供进入游戏的唯一入口。
2. **主页面与互动地图 (Main Map UI)**：
   - 取消传统的底部导航栏，将【背包】与【百科】功能置于**侧面按钮**。
   - 【罗盘】作为核心常驻道具，**固定置于左上角**（尺寸适中，便于边走边看）。
   - 地图上的玩家体现为**可移动的小人图标**，模拟真实走动感。
   - 采用**“点击地图目标点”**替代真实的 GPS 围栏检测，用于快速触发关卡剧情（测试/验证专属）。
3. **序章 (Prologue)**：古信件展开动画、初始道具（族谱残页/放大镜/罗盘）的获取表现、胡老夫子初次引导对话（透明背景单独人物立绘+气泡）。
4. **第一幕 (Chapter 1 - 胡文光牌楼)**：
   - 场景对话与背景讲解。
   - “牌坊形制”填空解谜（配合放大镜的自动高亮微动效）。
   - 拍照打卡并生成叠字图片（视觉占位或基础调用）。
   - 结算并颁发“仁”字明经印动画。

### **Out of Scope (不涉及/未来做)**
1. **真实 GPS 坐标围栏**：本次直接屏蔽复杂的 GPS 定位与纠偏逻辑，纯靠点击触发。
2. **后续所有关卡内容**：第2幕至终章、抛绣球等支线剧情、当地特产收集（仅作背包界面的占位视觉，不可交互）。
3. **复杂的成就与称号系统**：当期不做复杂的积分换算和称号发放逻辑。
4. **底层用户账号与存档**：无需考虑断线重连、复杂的微信授权与云端持久化，纯前端走流程即可。

## 4. 核心功能详述 (Functional Requirements)

### 4.1 入场与主界面模块
- **入场页**
  - **触发条件**：用户首次打开小程序。
  - **预期行为**：展示全屏静态或微动效美术背景，居中显示“进入西递秘档”按钮，点击后跳转至地图主界面。
- **主界面 (互动地图)**
  - **触发条件**：进入地图页。
  - **预期行为**：
    1. 渲染主视图为二维地图素材。
    2. 左上角常驻渲染【简易罗盘】小图标组件。
    3. 屏幕侧边（左/右侧中下部）常驻【背包】、【百科】悬浮按钮。
    4. 地图中心存在“玩家小人”图标。地图上显示“村口”与“胡文光牌楼”的可点击标记，后续建筑仅显示灰色锁定状态。

### 4.2 序章模块 (剧情引导)
- **触发条件**：玩家首次进入主页面后自动弹出。
- **预期行为**：
  1. 播放“古信展开”动画并显示文字。
  2. 弹出道具获取弹窗（展示族谱残页、放大镜、罗盘）。
  3. 左侧/底部切入【胡老夫子】NPC 立绘（透明背景、单独人物，无场景背景）及对话气泡。
  4. 交互：用户点击“继续”或“准备好了”完成序章，NPC 离场。

### 4.3 第一幕模块 (胡文光牌楼)
- **触发条件**：玩家在地图上点击“胡文光牌楼”地标。
- **预期行为**：
  1. 切入胡老夫子/胡文光 NPC 对话（NPC立绘须为透明背景的单独人物，不含场景背景）。
  2. **图片找茬/填空组件**：展示牌楼大图，底端提供“（ ）柱（ ）间（ ）楼”输入框。用户答题。
  3. **放大镜动效**：停留超过 x 秒或答错时，触发指定图块的高亮闪烁。
  4. **拍照打卡**：调用系统相机或调起本地相册，上传照片后，叠加两句固定对联文案（视觉表现合成）。
  5. **结算奖励**：弹出金色大印章（“仁”字）动效，并在后台更新百科解锁状态。

### 4.4 侧边辅助功能 (占位与展示)
- **百科面板**
  - **触发条件**：点击侧边【百科】按钮。
  - **预期行为**：弹出半层或全屏面板，列表化展示条目。对于通关第一幕的用户，将相关百科（牌坊形制等）标为高亮可点击阅读，其余条目标为灰色。
- **背包面板**
  - **触发条件**：点击侧边【背包】按钮。
  - **预期行为**：展示九宫格或列表，其中“基础道具”区点亮族谱、放大镜、罗盘；“明经印”区点亮仁字印；其余为空。
