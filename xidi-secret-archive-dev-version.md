# 《西递秘档：明经遗梦》
# 普通模式 · 开发交付版

> **游戏定位**：基于微信小程序的开放式实景解密探索游戏，以西递古村落为线下场景载体，以「游戏化叙事+沉浸式导览+轻量收集」为核心模式。
> 
> **适用对象**：普通游客、亲子家庭、初次到访者。无需任何专业知识背景。
> 
> **主线时长**：约1.5-2小时，可分多次完成。
> 
> **技术平台**：微信原生小程序或uni-app框架，支持iOS/Android。

## 一、改造原则与核心理念

### 1.1 核心理念

- **线上辅助线下，不繁琐**：小程序仅提供引导和验证，探索行为发生在线下。单关时长控制在10-15分钟。
- **容错友好**：答错不卡关，连续答错两次后系统直接给出提示。
- **无AR也可完整通关**：所有需要扫描触发的内容，均提供「拍照上传」或「手动选择」的替代方案。
- **开放式探索**：推荐路线为从南至北，但玩家可自由选择。跳过某关直达后续关卡时，NPC会友善提醒「尚有未解之缘」，但不会强制回头补完。

### 1.2 改造要求对照表

| 改造要求 | 具体策略 |
|---------|---------|
| 三维度融合 | 以「寻访者」单一身份进入，每关自然融入宗族文化、历史文物、建筑风水三大元素 |
| 降低难度 | 取消复杂计算题；取消严格排序题；每题答错两次后提供选项提示；所有谜题以「观察+轻量推理」为主 |
| 缩减关卡 | 序章+6幕实地关卡+终章，共8个节点 |
| 游览顺序 | 严格参考景区导览图，按「村口→前街→横路街→村中心→正街→村南→观景台」的游客自然动线设计 |
| 减少文言文 | NPC对话采用现代白话口语；所有谜题提示用现代汉语 |
| 补充玩家视角 | 每关增加「玩家视角」板块，从第一人称描述行走动线、视觉感受和操作步骤 |
| 完整可交付 | 故事线完整讲完，角色对话全部列出；百科10条；所有系统模块完整保留核心功能 |

## 二、世界观与主线故事

### 2.1 时空设定与核心线索

- **主时空**：2026年的西递古村落，玩家以「寻访者」身份进入。
- **核心设定**：西递村中流传着古老传说——胡昌翼在临终前留下一份「明经秘档」，记载了李唐皇室与胡氏家族世代守护的精神遗产。这份秘档被分成六个碎片，分别藏在村中六处核心建筑之中，需要集齐六枚「明经印」才能开启最终真相。
- **历史回响机制**：玩家在特定地点触发虚拟NPC的回忆叙述，聆听历史人物讲述当年的故事。

### 2.2 玩家身份（普通模式统一设定）

你是一名普通的文化爱好者，在社交媒体上收到了一封匿名私信，附有一张泛黄老照片——照片上是西递的胡文光牌楼，背面写着一行字：「六堂归一，秘档自现。明经胡氏，待有缘人。」出于好奇，你来到了西递村。

**专属道具**：进入游戏后，胡老夫子会交给你三件物品——
- **族谱残页**（遇到祠堂、牌位时自动高亮关键信息）
- **放大镜**（观察三雕细节时自动高亮可疑区域）
- **简易罗盘**（在建筑格局分析时显示大致方向提示）

这三件道具是系统自动发放的，玩家无需选择身份，在对应谜题中自动生效。

## 三、西递小百科

### 3.1 百科入口设计

- **入口位置**：小程序底部导航栏常驻「百科」按钮，任何页面均可一键进入。
- **展示形式**：
  - 已解锁内容：彩色图标+完整文字，可展开阅读
  - 未解锁内容：灰色图标+「通关第X幕后解锁」提示
  - 每个条目支持「收藏」「分享」「朗读」（语音播报）
- **与游戏的结合**：解谜过程中若连续两次错误，系统提示「是否需要查看相关百科知识？」

### 3.2 百科内容架构（10条）

| 类别 | 条目名称 | 解锁条件 |
|------|---------|---------|
| 建筑营建（2条） | 四水归堂的天井智慧 | 通关第四幕后解锁 |
| | 马头墙的五阶形制 | 在村内拍摄3张马头墙后解锁 |
| 三雕艺术（2条） | 石雕鹤鹿龙虎的象征 | 通关第一幕后解锁 |
| | 漏窗图案的寓意 | 通关第三幕后解锁 |
| 楹联文化（2条） | 瑞玉庭楹联的缺笔设计 | 完成第二幕拍照打卡后解锁 |
| | 履福堂楹联的断句哲学 | 完成第六幕拍照打卡后解锁 |
| 宗族历史（2条） | 「真李假胡」的改姓渊源 | 通关第一幕后解锁 |
| | 胡文光的宦海生涯 | 通关第一幕后解锁 |
| 民俗风物（2条） | 抛绣球的婚俗源流 | 通关第五幕后解锁 |
| | 徽派建筑的选址风水 | 通关第五幕后解锁 |

**单条条目格式**：每条控制在300-500字，包含「一句话总结」「详细解读」「趣味冷知识」三个板块，配以高清图片或线描插图。文字风格轻松有趣。

**示例：百科条目「四水归堂」**
- **一句话总结**：天井是徽派建筑的心脏，雨水从四面屋顶流入家中，寓意「肥水不流外人田」。
- **详细解读**：徽商常年在外经商，家中多为妇孺老幼。天井不仅采光通风，更承载着家族团聚的象征意义。下雨时，四面的雨水汇聚到天井中央的排水口，经暗沟流入自家池塘，既是实用设计，也是「聚财守家」的风水理念。
- **趣味冷知识**：西递有些老宅的天井排水口做成铜钱形状，称为「财源滚滚」。雨水流经铜钱孔洞的声音，在安静的老宅中格外清脆，古人称之为「听财声」。

## 四、以地图为核心的探索机制

### 4.1 游戏化地图设计

- **探索迷雾**：玩家实际行走在西递村中时，GPS定位自动驱散周围迷雾，无需手动操作。
- **已探索标记**：走过的新巷道自动标记为「已探索」，获得小额积分奖励（每条巷道5积分）。
- **成就系统**：探索全村30%/60%/100%区域，分别获得「巷陌行者」「迷宫旅人」「西递通」称号。
- **离线兼容**：GPS信号弱时，玩家可手动点击「我已到达此处」来推进，但需回答一个验证性问题（如「你看到了什么颜色的门楼？」）。

**地图UI简化**：
- 仅显示三类标记：①主线建筑（金色图标）②拍照打卡点（红色相机图标）③特色触发点（蓝色星标）
- 点击任意标记，弹出简短说明和距离提示（如「瑞玉庭·商字之谜，向北步行约80米」）
- 不提供复杂的路径规划，仅指示大致方向，鼓励玩家自主探索巷道

### 4.2 六个文化锚点与自由路线

## 五、拍照打卡系统

### 5.1 拍照任务设计原则

- **轻量不繁琐**：每处拍照任务控制在1分钟内完成。
- **与建筑绑定**：每处打卡均选自该建筑的标志性景观。
- **地图精确标注**：每个拍照打卡点在小程序地图上以红色相机图标标记。

## 六、虚拟NPC体系

### 6.1 核心虚拟NPC

全部通过立绘+音频+文字对话呈现，取消真人演员依赖。

| NPC名称 | 身份 | 功能 | 出现地点 | 形象风格 |
|---------|------|------|---------|---------|
| 胡老夫子 | 胡氏宗族长老，游戏引导者 | 发布主线任务、提供背景讲解、终章揭晓真相 | 村口牌楼（首次出现）、敬爱堂、终章观景台 | 白须老者，手持竹杖，身着青色儒衫 |
| 胡文光 | 明代刺史，历史人物角色化 | 讲述牌坊建造故事、发布石雕谜题 | 胡文光牌楼 | 明代官服，正色端坐，神情威严而亲和 |
| 绣楼小姐 | 清代大家闺秀（虚拟角色） | 在绣楼下与玩家对话，讲述婚俗文化 | 大夫第绣楼 | 清代闺秀装束，手持团扇，立于绣楼窗前 |
| 胡贯三 | 清代巨商（虚拟角色） | 讲述追慕堂修建故事、发布门神谜题 | 追慕堂 | 富态商人装扮，手持账簿，笑容可掬 |

**绣楼互动说明**：原方案中「真人NPC抛绣球」改为虚拟互动——玩家在绣楼下与虚拟绣楼小姐对话后，小姐「抛出」一枚虚拟绣球（屏幕动画），玩家点击接住即可获得「喜」字印章。小姐会讲述徽州婚俗「抛绣球择婿」的历史渊源。

### 6.2 NPC触发机制

| 触发方式 | 技术实现 | 适用场景 |
|---------|---------|---------|
| GPS围栏触发 | 进入建筑周围20米范围，自动弹出NPC对话气泡 | 村口胡老夫子首次出现、各建筑入口欢迎语 |
| 手动点击触发 | 玩家在小程序地图上点击NPC图标，主动发起对话 | 已解锁NPC的回访对话 |
| 任务进度触发 | 完成特定谜题后，NPC自动弹出给予下一步指引 | 每幕谜题完成后的剧情推进 |

**音频设计**：每个NPC配备5句核心语音（总时长不超过2分钟），采用古风配乐背景。语音内容为关键剧情点和谜题提示，非关键文字对话以文字形式呈现。

## 七、解谜机制矩阵

### 7.1 四大解谜类型

| 解谜类型 | 实现方式 | 应用关卡 |
|---------|---------|---------|
| 图像识别 | 玩家拍摄建筑照片上传，云端识别或人工审核；或提供高清参考图，玩家在图内点击目标元素 | 瑞玉庭「商」字识别、漏窗图案识别 |
| 雕刻扫描 | 提供高清参考图，玩家在图内圈出目标元素 | 牌楼石雕动物识别、「孝」字拆解 |
| 方位导航 | 基于GPS的虚拟地图标记，玩家在地图上点击标注 | 西园四君子方位推理 |
| 机关解谜 | 纯屏幕内虚拟机关盘操作 | 青云轩地风井方位调整 |

### 7.2 难度控制措施

1. **观察题**：提供高清大图，可疑区域用微弱闪光提示（3秒后自动闪烁）。
2. **排序题**：祖先排序仅3人（胡昌翼→胡文光→胡贯三），每人下方有明确朝代标签。
3. **计算题**：取消数字密码计算，改为直接观察建筑结构得出答案。
4. **记忆题**：取消时间限制，允许反复查看原图。
5. **文字题**：生僻字自动加注拼音和释义。

- **辅助机制**：答错两次后，动物图标下方浮现文字提示（如「鹤：鹤活得久，因为它不伤人，就像心地善良的人被人尊敬」）。

### 终章：秘档真相（村尾观景台/荷花池，10分钟）

#### 【触发方式】

#### 【完整剧情与对话】

> **胡老夫子**（站在观景台上，面向西递全景，背影苍老而庄重）：
> 「你来了。站在这里，整个西递尽收眼底。你看这白墙黛瓦、马头墙、青石板路，这就是我们胡氏几百年积攒下来的家业。但家业会旧，房子会老，什么东西不会老呢？」

> **玩家选项A**：「是精神？」
> **玩家选项B**：「是文化？」
> **玩家选项C**：「我不确定。」

**（玩家进入印章排列界面）**

六枚明经印需要按正确顺序排列在屏幕上的六个空格中：
1. 「仁」字印（第一幕·胡文光牌楼）
2. 「义」字印（第二幕·瑞玉庭）
3. 「礼」字印（第三幕·桃李园/西园东园）
4. 「孝」字印（第四幕·敬爱堂）
5. 「巧」字印（第五幕·大夫第/绣楼/青云轩）
6. 「和」字印（第六幕·追慕堂/履福堂）

- **简化**：印章下方有获取地点标签，玩家只需按照游览顺序依次拖拽即可。排错两次后，地点标签闪烁提示正确顺序。

**（印章排列完成，屏幕浮现一卷泛黄的族谱/秘档）**

> **胡老夫子**：「这便是『明经秘档』。你打开看看。」

**（屏幕缓缓展开秘档内容，显示胡昌翼留下的家训六则）**

**明经秘档·家训六则**：
1. 孝顺长辈
2. 好好读书
3. 讲义气、守信用
4. 吃亏是福
5. 懂得退让
6. 勤俭持家

> **胡老夫子**（声音温和而坚定）：
> 「看到了吗？没有金银珠宝，没有藏宝图，只有这六条家训。这就是先祖留给后人的全部『秘档』。真正的宝藏，不是埋在地下，是活在人的心里。你今天走过了牌坊、老宅、书馆、祠堂、园林、厅堂，每一步都在学这些道理。」

> **胡老夫子**：「你已经证明自己是明经胡氏的有缘人。这份精神遗产，从今以后也归你所有。带回去，讲给你的家人、朋友听。只要还有人记得，这份秘档就永远不会消失。」

**（屏幕绽放金色光芒，胡老夫子的形象渐渐淡去）**

> **系统提示**：「恭喜通关！您已获得『明经有缘人』称号！」

#### 【终章额外内容】

- **特产收集**：若玩家在终章时收集了3件特产道具，胡老夫子会额外赠予「徽味行者」专属感谢语：「你不仅懂道理，还懂生活。徽州的味道，也是秘档的一部分。」
- **人物志解锁**：若集齐了3块名人轶事碎片，则额外解锁「明经知音」专属对话：「你了解了胡昌翼、胡贯三、胡文照的故事，你已不仅是寻访者，更是知音。」

**（终章完成，主线通关）**

**系统奖励**：
- 「明经有缘人」专属数字徽章
- 通关纪念海报（可分享至朋友圈）
- 「我的西递足迹」回顾页面（展示走过的路线、解开的谜题、收集的道具）
